Ashirwad Orna - B2B Jewelry Components
A sleek, high-performance static website built for a premium Indian B2B jewelry manufacturer. Designed to showcase bulk jewelry findings, precision screws, and support posts to retail jewelers and Karigars.

Key Features:

💎 Premium UI/UX: High-end aesthetic using Playfair Display, deep obsidian/gold themes, and ultra-smooth scroll-reveal animations.
📱 Mobile-First & Fully Responsive: Edge-to-edge mobile navigation and flawlessly scaling fluid layouts.
🖼️ Dynamic Product Galleries: Vanilla Javascript-powered product listing grids and an interactive full-screen image lightbox slider.
🚀 Highly Optimized: 100% SEO-ready with lazy-loading, async decoding, pre-connected fonts, and comprehensive OpenGraph/Twitter meta tags.
🛠️ Tech Stack: Vanilla HTML5, Javascript, Tailwind CSS (CDN), and Phosphor Icons.
